Bezier Curves, Polynomials and Gradient Descent

This is a demonstration using modern machine learning to find the "control points" of a Bezier Curve.

The curve on the left is described by the rainbow colored control points.

Given the resutling curve, this module uses gradient descent to find those same control points. It could solve for fewer or more control points, as many as needed to fit the curve. It could also solve for more than the two dimensions of a typical bezier curve.
