Bezier Curves, Polynomials, Gradient Descent

Draw a curve by moving the control points below, then click train. Given the resutling curve, this module uses gradient descent to find the control points. It could solve for fewer or more control points, as many as needed to fit the curve. It could also solve for more than the two dimensions of a typical bezier curve.
